# Blockchain Spare Parts Management system

Autor: Natalia Pieczko

Repozytorium jest częścią pracy inżynierksiej.

## Instrukcja uruchomienia

Aby uruchomić projekt należy wykonać następujące kroki:

1. Sklonować repozytorium z kodem źródłowym za pomocą polecenia:
```
git clone https://github.com/nataliap203/Blockchain-Spare-Parts-Management.git
```
2. W oknie terminala przejść do głównego katalogu projektu i zbudować obrazy Docker:
```
cd Blockchain-Spare-Parts-Management
docker-compose up --build
```
3. Otworzyć przeglądarkę internetową i przejść na adres http://localhost:8501, aby
uzyskać dostęp do aplikacji klienckiej.
4. Aplikacja w lewym górnym rogu ekranu umożliwia wybór wersji systemu (Ethereum
lub VeChain).
